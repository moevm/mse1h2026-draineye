# Название проекта

Backend-сервер для мобильного приложения DrainEye.

## Требования

- Python 3.11 или выше
- pip (менеджер пакетов Python)
- Доступ к интернету для установки зависимостей
- Аккаунт Firebase (создайте проект в [консоли Firebase](https://console.firebase.google.com/))
- Аккаунт Cloudinary ([сайт](https://cloudinary.com/))

## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd <имя-папки-проекта>
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
# или
venv\Scripts\activate     # для Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройка Firebase

Необходимо в настройках проекта включить Cloud Firestore:
- В левом меню выберите **"Firestore Database"**
- Нажмите **"Создать базу данных"**
- Выберите режим **"Тестовый режим"** для разработки

В консоли Firebase откройте **Настройки проекта** 
- Перейдите на вкладку **"Сервисные аккаунты"**
- Нажмите кнопку **"Создать новый закрытый ключ"**
- Подтвердите действие — скачается JSON-файл с ключами

Настройка Authentication:
- В левом меню выберите "Authentication".
- На вкладке "Sign-in method" включите провайдер "Email/Password" и Gmail.

### 5. CLoudinary

- Создайте аккаунт
- В панели управления (Dashboard) найдите следующие данные: Cloud Name, API Key, API Secret
  (Эти данные понадобятся для заполнения файла .env.)

### 6. Создайте и настройте файл .env

В корне проекта уже есть файл `.env.example` с примером конфигурации. 
Скопируйте его в `.env`:

```bash
cp .env.example .env
```

Отредактируйте файл `.env`, настроив свои параметры:

```bash
nano .env
# или используйте любой текстовый редактор
```

### 7. Запустите проект

```bash
python main.py
```

### 8. Для проверки с приложением и веб-сайтом спользовался tuna.am

- Установка:
```bash
curl -sSLf https://get.tuna.am | sh
```
- Сохранение токена (токен уникальный):
```bash
tuna config save-token <token>
```
- Запуск:
```bash
tuna http 8080
```
